鲲鹏迁移工具部署

华为云购买服务器

  • 购买配置为：
    鲲鹏计算；
    AI加速型；
    公共镜像版本：CentOS 7.6.64

下载鲲鹏代码迁移工具

鲲鹏代码迁移工具下载-鲲鹏社区
选择2.5.0鲲鹏服务器下载

连接XShell操作

连接成功后，进入opt目录执行命令：
yum -y install lrzsz

安装rz工具

安装成功后上传鲲鹏代码迁移工具压缩包

解压安装

解压

执行tar -zxvf 压缩包名进行解压

解压成功后进入解压完成的文件中

安装

执行./install web运行文件中的install脚本(已web的形式)

安装过程中出现需要按y、n或回车（均按y或回车）

安装完成如下图：

防火墙

打开防火墙：
systemctl start firewalld

查看防火墙是否打开：

systemctl status firewalld

查看8084端口是否打开（no表示没有打开）：

firewall-cmd --query-port=8084/tcp

打开8084端口：
firewall-cmd --add-port=8084/tcp –permanent

重新加载配置：
firewall-cmd –reload

再次查看8084端口是否打开（yes表示已经打开）
firewall-cmd --query-port=8084/tcp

访问

访问
https://你的公网ip:8084/porting/#/login
https://123.249.29.100:8084/porting/#/login
成功。
